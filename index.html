Hello! Welcome to the "Roll and Win" Game. This game contains two players, game starts with player1. Initially Player 1 rolls the dice and if number on the dice is other than '1' then the number will be added to 'currentScore'. When player clicks the Add button the currentScore will be added to totalScore. Game control goes to Player 2 when the value on dice is '1' or Player 1 clicks the Add button. Similiarly this continues for Player 2 also. When any Player scores more than or equal to '50' he wins and game will finish.
